# Energy Time Series Forecasting in R
This repostory contains time-series forecasting project using R.

The model analyse energy data from Germany and apply statistical forecasting mothods.

# Methhods used:
- ARIMA modelling
- Seasonal differencing
- ACF/PACF analysis
- Ljung-Box tests
- Forecast evaluation

# Tools:
R, forecast package, tseries, dplyr
